# FastLacellsGenerator
Lacells.db is automatically building using github actions

### How to use

install apk from this [link](https://gitlab.com/-/project/23195867/uploads/3e5994177ce852bb7bf5b74743b6d99b/LocalGSMBackend.apk) and place lacells.db to /sdcard/Android/data/org.fitchfamily.android.gsmlocation/files/lacells.db
Note: You need to Install UnifiedNlp from this [link](https://github.com/microg/UnifiedNlp/releases/download/v1.6.8/UnifiedNlp.apk) and integrate it to system via magisk [module](https://github.com/misaka4e21/magisk-unifiednlp-aosp-enabler)
